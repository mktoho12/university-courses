# 対話ノート①：シェアハウスの掃除問題から考える「公共」

第1章（はじめに/公共哲学とは何か）の内容を使った、初めての対話。
題材はユーザー自身の生活＝シェアハウスの共有スペース掃除・ゴミ出し。

## きっかけの具体例（ユーザー提供）
- シェアハウスで、共有スペースの掃除・ゴミ出しを積極的にやる人とやらない人に分かれる。
- 掃除は手間（コスト）。やらずに親切な人に任せれば個人はトクをする。
- 自分がやると手間を払うが、全体の利益にはなる。
- ユーザー自身は「やる側に近い」。

## この題材が当てた哲学的構造
- **フリーライダー（ただ乗り）問題 / 共有地（コモンズ）の悲劇**：
  各自が個人的に合理的な選択をすると、全体では最悪の結果になる。
- 第1章 section3 の図（国家⇔社会＝公共⇔個人）の、ミニチュア版そのもの。
  「私的利益と公的利益の衝突」を生活で体験している。

## 対話の流れ（思考がどう動いたか）

1. **「やらない人」を steelman する練習**（AIが"やらない住人"を演じた）。
   - 住人の主張：「きれいさの基準は人それぞれ。君が自分の基準で勝手にやってモヤッとしてるだけでは？」
   - ユーザーの応答：基準の相対性を**認め、自分にも跳ね返した**
     （「自分の掃除も、もっと基準の高い人にはサボりに見えるかも」）。
     → 立場の交換可能性（自分にだけ都合よく使わない）を実践。

2. **「基準は人それぞれ」だけでは片付かない、では一線は引けるか？**
   - ユーザー：「**一線は引けない**。明文ルールで全レアケースを縛るのは非現実的。
     実際には暗黙の良識で回り、たまに合わない人が出ても話し合い／時に退去で何とかなっている」。
   - これは政治哲学でいう **自生的秩序（ヒューム/ハイエク）／コミュニタリアニズム的**な立場。
     AIが誘導しかけた「合意・契約で線を引く」（契約論/リバタリアニズム寄り）に、
     ユーザーが無自覚に**対抗する立場**を取った＝自分でひとつの立場に立った瞬間。

3. **「同調圧力では？」という揺さぶり**（AIが再び住人を演じた）。
   - 「明文ルールがない＝多数派のなんとなくの感覚に従わない人を勝手に"モンスター認定"して
     追い出しているだけ。それは優しい秩序か、ただの同調圧力か？」
   - ユーザーの応答（この対話の山場）：
     - 批判を認めつつ、自分の立場に**限界条件**を付けた
       （「暗黙の良識が大きく異なる人に全部合わせると全体が崩壊する」「シェアハウスは
         営利を考えないと維持できない」）。
     - さらに**自分から言葉を訂正**：「モンスター・追放という言い方はよくなかった。
       **うちのコミュニティに合わないだけで、その人が悪い・劣っているわけではない**」。
     → 「人格の優劣を裁く」から「相性・適合の問題」へ判断の種類を切り替えた。
       ＝ section3「異なる価値観をもつ人をリスペクトする」の実践。

4. **残った緊張＝今日の核心**：
   - 「その人を劣っているとは言わない（尊重）」のに「共同体から出てもらうことはある（排除）」。
   - **尊重と排除の同居** ＝ コース第12章「公共性と排除」のテーマに、
     身近な掃除の話から自力で到達した。

## ユーザーの価値軸の変化（before → after）

- **before**：掃除をしない人＝みんなのために行動できない、ズルい人。
- **after**：その人にも言い分がある／うちのシェアハウスの感覚と合わないだけ。
  "悪い・劣ってる"イメージは頭をよぎるが、同じ共同体に属するのが難しいだけで、
  その人を尊重できないわけではない。

→ 立場（自分は"やる側"でいい）は変わらず、**解像度が上がった**。
　 これが教材 section4「対話を通じて自分の価値軸を一貫して進化させ続ける」の一回分。
　 ※「嫌な感情がよぎるのを認めたうえで、それでも尊重を選ぶ」を自分の言葉で言えた点が良い。

## 次につながる伏線（後の章で回収したい）
- 「the least worst option」「フリーライダー問題」への対処
  → 功利主義（第4章）：全体の幸福を最大化するなら掃除当番を強制すべき？
  → ロールズ（第5章）：「無知のヴェール」をかぶったら（自分がどの性格の住人になるか
     分からないとしたら）どんな掃除ルールを選ぶ？
  → コミュニタリアニズム（第6章）：ルールより共同体の慣習・徳。今日ユーザーが立った立場。
  → リバタリアニズム（第7章）：誰も強制されない。やりたい人がやればいい。
  → 公共性と排除（第12章）：尊重と排除の同居問題。
- 次に同じ題材を使うなら「無知のヴェール」で掃除ルールを設計してみると面白い。
