# 公共哲学

大学の授業「公共哲学」を、AIと相談しながら進めるための作業フォルダ。

## この科目で自分がどうなりたいか（学習のゴール）

> **異なる立場・考え方を持つ人との対話や協働をスムーズにできるようになること。**

そのために身につけたいのは、主に次の力：

1. **複数の立場を自分の中で行き来できる**（最優先）
   功利主義者ならどう言うか / リバタリアンなら / コミュニタリアンなら…を、自分の中で複数の声として再現できる。意見の違う相手が「どの原理から喋っているのか」が見える状態。
2. **概念で世界を切り分けられる**（できれば、副産物として）
   ニュースや日常を見て「これは帰結主義 vs 義務論の対立だな」と名前を付けて考えられる。優先はしない。

### 評価基準（「学んだ」と言える状態の定義）

テストで正解できることではなく、

- **立場の違う相手の理屈を、相手が納得する形で再現できるか**（＝ steelman できるか）
- その上で、自分の考えを持ちつつ反論にも応答できるか

を「学べた」の目安とする。
※ なぜこの基準なのか：目的が「対話・協働」だから。点が取れること(帰結＝功利主義的評価)より、
　 ものの考え方という自分の能力が育っているか(ケイパビリティ的評価)を重視する。
　 ——この評価基準を考えること自体が、すでに本科目の練習になっている。

## 進め方

- **ペース**：実際の授業の進度に合わせる。今週やる章を、その都度ここで一緒に深掘りする。
- **AIの役割**：
  - 各トピックを解説する／理解度を問う問いを出す
  - ときどき**「あなたと違う立場の人」を演じる**。論破ごっこではなく、その人を理解して応答する練習をする。
  - 学んだ内容を復習用ノートにまとめる
- **科目間の共通化はしない**。この科目はこの科目に合った形で進める。
