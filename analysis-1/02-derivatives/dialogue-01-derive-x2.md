# 対話ノート①：$(x^2)' = 2x$ を定義から自力導出＋極限の本質

第2章前半。第1章で「結果だけ借りた」$x^2 \to 2x$ を、今度は微分係数の定義から自分の手で証明した。
方式は前回同様：1問ずつ、AIは答えを言わずヒントと方向づけのみ、本人が言語化。

## 導出の流れ（すべてユーザーが出した）

道具：$f'(a) = \lim_{h\to0}\dfrac{f(a+h)-f(a)}{h}$ のみ。$f(x)=x^2$ を代入。

1. 代入：分子 $f(a+h)-f(a) = (a+h)^2 - a^2$
2. 展開：$(a+h)^2 = a^2 + 2ah + h^2$（中学の $(x+y)^2$ パターンを自力適用）
3. 整理：$a^2+2ah+h^2-a^2 = 2ah+h^2$ → **さらに $h$ でくくって $h(2a+h)$**（先を見越した変形。後の約分のため。自発的にここまで行った＝白眉）
4. 約分：$\dfrac{h(2a+h)}{h} = 2a+h$（3でくくったおかげで $h$ が消しやすかった）
5. 極限：$\lim_{h\to0}(2a+h) = 2a$
6. 結論：$f'(a)=2a$、点 $a$ を動かして $f'(x)=2x$

$$f'(a) = \lim_{h\to0}\frac{(a+h)^2-a^2}{h} = \lim_{h\to0}\frac{h(2a+h)}{h} = \lim_{h\to0}(2a+h) = 2a$$

## 極限の本質を自力で言語化（今日の核心）

問い：「約分で $h$ で割ったのに、最後は $h\to0$。0で割れないはずでは？ 矛盾しない理由は？」

ユーザーの回答（ほぼ完璧）：
> 0で割ることはできないけど、割る数をどこまでも0に近づけていくことはできる。
> 極限の lim が言ってるのは「$h$ が0に近づくとき $2a+h$ はどこに向けて近づいていくか」なので、**$h$ は一度も0になっていない**。

- 「近づく」と「なる（代入）」を自力で区別できた。これは多くの人が lim を代入と誤解してつまずく所。
- ここが分かっていると極限・連続・微分が全部すっきり乗る。
