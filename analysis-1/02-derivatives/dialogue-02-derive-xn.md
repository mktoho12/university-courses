# 対話ノート②：$(x^n)' = nx^{n-1}$ を二項定理で一般化

第2章前半。対話①の $x^2$ を一般の $x^n$ に拡張。同じ5ステップ（代入→展開→整理→約分→極限）を二項定理で。
方式は継続：1問ずつ、AIは答えを言わずヒント/方向づけ、本人が言語化。

## 導出の流れ（すべてユーザーが出した）

道具：定義 $f'(a)=\lim_{h\to0}\frac{f(a+h)-f(a)}{h}$ ＋ 二項定理。

二項定理：$(a+h)^n = a^n + n a^{n-1}h + \binom{n}{2}a^{n-2}h^2 + \cdots + h^n$

0. **検算**：二項定理に $n=2$ を入れて今日の $a^2+2ah+h^2$ と一致を確認。
   ユーザーは「真ん中の省略項 $\binom{2}{2}a^0h^2$ と末尾 $h^n$ が同じ $h^2$ を指す（$n=2$は項が畳まれて見える）」まで自力で気づいた。
1. 代入：分子 $(a+h)^n - a^n$
2. 展開：二項定理
3. 整理：$a^n$ が相殺 → $n a^{n-1}h + \binom{n}{2}a^{n-2}h^2 + \cdots + h^n$
4. $h$ でくくる：$h\left(n a^{n-1} + \binom{n}{2}a^{n-2}h + \cdots + h^{n-1}\right)$
   （各項の $h$ の個数が1ずつ減るのを正確に追えた）
5. 約分：$n a^{n-1} + \binom{n}{2}a^{n-2}h + \cdots + h^{n-1}$
6. 極限 $h\to0$：$h$ を含む項は全消滅 → $n a^{n-1}$
7. 結論：$f'(x) = n x^{n-1}$（冪の微分公式）

$$f'(a) = \lim_{h\to0}\frac{(a+h)^n - a^n}{h} = \lim_{h\to0}\frac{h(n a^{n-1}+\binom{n}{2}a^{n-2}h+\cdots+h^{n-1})}{h} = n a^{n-1}$$

## 今日の核心：「生き残る項」のふるい

セッション冒頭でAIが予言「$a^2$でも$h^2$でもない、$h$が1個の項だけが生き残る」→ 実際に的中。

生き残った項 $n a^{n-1}h$ は **$a$ が $(n-1)$個、$h$ がちょうど1個**。
ユーザーが「なぜこれだけ生き残るか」を自力で完璧に言語化：
> $h$ が2個以上のものは(極限で)消えるし、$a$ だけのものは引き算で消えてしまうから

| 項のタイプ | 運命 | 理由 |
|---|---|---|
| $a$ だけ（$a^n$） | 引き算で消える | $-a^n$ と相殺 |
| $h$ が2個以上 | 極限で消える | 約分後も $h$ が残り $h\to0$ で 0 |
| $h$ がちょうど1個 | **生き残る** | 約分で $h$ が消え、$h$ なしの項に |

→ **微分の本質＝「$h$ が1次（ちょうど1個）の項を取り出す操作」**。第2章2-3「一次近似」$f(a+h)\approx f(a)+f'(a)h$ の $h$ が1個の項、に直結する伏線。

（補足：ユーザーは最後に「$a$と$h$が1個ずつ」とも言ったが、これは $n=2$ 限定。一般には $a$ は $(n-1)$個。最初の答え「$a$ が $n-1$個、$h$ が1個」が正確。両方の発言を合わせると完璧だった。）
