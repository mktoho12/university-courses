# 対話⑨ 微分の線形性（2-4）

教材: `material.md` の「s4『微分の線形性』」節。例題2-4-1, 2-4-2／練習問題2-4-3。

## 流れ
1. 理屈から入った（和の一次近似を足す→hでくくる→係数を取り出す）が、最後の
   「(f+g)′(a) は何に等しい?」で本人が「ぐちゃぐちゃする」と停止。
   → 方針転換：理屈を置いて**手を動かすドリルへ**（本人要望）。
2. 自作ドリル10問（和・定数倍・定数項消滅・符号 −cos x→+sin x・分数係数）を**全問正解で完走**。
   - 詰まりは2か所だけで、どちらも本人が自力修正：
     - (x⁴)′ を 3x³ と書く → 「4x³でした」と自己訂正（指数を下ろす操作）
     - 最終問 f=1+x+½x²+⅓x³+¼x⁴ → 「0+1+x+x²+x³ だから 1+x+x²+x³ だ！」と自力到達
   - 教材の例題2-4-2・問題2-4-3 をドリル内で自力クリア。
3. 例題2-4-1（x²+sin x の導関数 = 2x+cos x）を本人が指摘して取りこぼし回避→自力クリア。
4. 理屈に再挑戦（本人選択「軽く挫む」）。具体例 f=x², g=x³ で
   **道A(先に足して微分)=道B(先に微分して足す)=2x+3x²** が一致するのを確認。

## ⭐ 今日いちばんの収穫（本人の自己診断）
本人が詰まりの正体を正確に言い当てた：
「線形性の理屈じゃなく、**関数どうしを足して新しい関数 h を作る**というのが初耳で面食らった。
h(x)=x²+x³ がいきなり出てきた。教科書に説明はあるか?」
- → 教材本文に「関数の和の定義」の説明は**無い**（いきなり (f+g)′=f′+g′ を使っている）。
  本人の「教材に穴がある」感覚は正しい。
- 穴を埋めた：**f+g とは「同じ x を f にも g にも入れて、出た2数を足して返す機械」という"約束(定義)"**。
  定理ではない。x=2 で h(2)=f(2)+g(2)=4+8=12、x²+x³ に直接入れても 12 で一致 → 同じ機械の別の書き方。
- 本人が「道Aは当たり前に見える」と言ったのは深い：道Aで無意識にやる「項ごとに微分して足す」が
  許される保証こそが線形性の定理本体。「当たり前にしたのは本人」。

## 到達点
- 微分の線形性（和・定数倍）：**手は完全に自動化**。理屈も具体例で道A=道Bを確認。
- 前提の穴（関数の和の定義）を本人が自力で検知→足場を取って解消。挫折対策の理想形。

## 教訓（運用）
