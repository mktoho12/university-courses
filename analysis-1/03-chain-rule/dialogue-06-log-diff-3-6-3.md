# dialogue 3-6-3：対数微分 f(x)=√(x+2)(x+1)³（自力完走）

対数微分で $f(x)=\sqrt{x+2}\,(x+1)^3$ を最初から最後まで自力で通した。s6・ch03 完了。

## 流れ
- $f=ab$ の形と見て $\log f=\log\sqrt{x+2}+\log((x+1)^3)$ と分解。両辺を微分して骨組み $f'/f=(\sqrt{x+2})'/\sqrt{x+2}+3(x+1)^2/(x+1)^3$ をゼロから正しく書けた（$\log$ の分解も第2項の連鎖律も◎）。
- 唯一の詰まり：$(\sqrt{x+2})'$ を $-(x+2)^{-2}$ と置いた。「√って −1 乗だっけ？」の素の疑問から、**√ = 1/2 乗**、ついでに $x^{-1}=1/x$ を自分で確認し直し、$(\sqrt{x+2})'=\frac12(x+2)^{-1/2}$ に到達。
- 代入して
  $$f'(x)=\sqrt{x+2}\,(x+1)^3\left(\frac{\frac12(x+2)^{-1/2}}{\sqrt{x+2}}+\frac{3(x+1)^2}{(x+1)^3}\right)$$
  で完成。整理形（$=\frac{(x+1)^3}{2\sqrt{x+2}}+3\sqrt{x+2}\,(x+1)^2$）は任意なので追わず、$f'$ が出た所で気持ちよく区切った。

## メモ
- ウォームアップ：$\log(a^b)=b\log a$ 自力想起◎。
- 教材原文の第1項分子 $(x+3)^3$ は誤植・正しくは $(x+1)^3$（material.md の検算メモ通り）。今回は展開まで行かず未確認のまま区切ったが、答えとしては通っている。
