# ch03 s2「関数の合成」対話ログ（2026-07-09）

## 進めた内容（例題・問題を全完走）

- **例題3-2-1**：$f(x)=x+1$, $g(x)=x^2$ → $g\circ f=(x+1)^2$、$f\circ g=x^2+1$。自力正解。
- **問題3-2-2**：$f(x)=1/x$, $g(x)=x+1$ → $g\circ f=1/x+1$、$f\circ g=1/(x+1)$。自力正解。
- **例題3-2-3**：$y=\sin(x^2)$ → $f(x)=x^2$, $g(x)=\sin x$, $(g\circ f)$。自力正解。
- **例題3-2-4**：$y=(x^2+1)^3$ → 三つの合成 $f=x^2, g=x+1, h=x^3$, $(h\circ g\circ f)$。自力正解。
- **例題3-2-5**：$y=\exp(-x^2)$ → 最初 3段（$-x$ と $x^2$ 分割）で出したが、2段にまとめ $f=-x^2, g=\exp x$, $(g\circ f)$ に自力修正。
- **例題3-2-6**：$y=\sin^3 x$ → 一度 $f=x^3,g=\sin x$ と順序を逆にしたが、順番確認の足場で自力修正 $f=\sin x, g=x^3$, $(g\circ f)$。
- **問題3-2-7**：$y=(x^2+3)^4$ を三つの合成 → $f=x^2, g=x+3, h=x^4$, $(h\circ g\circ f)$。自力正解。
