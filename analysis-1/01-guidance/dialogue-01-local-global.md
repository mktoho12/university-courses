# 対話ノート①：local/global と「微分係数 vs 導関数」を散歩で掴む（試運転）

第1章ガイダンスの内容で、新しい学習方式の試運転をした初回セッション。
方式：教材を見せず（or 見てもいいが自分の言葉で）問いだけ。1問ずつ刻み、AIは答えを言わずヒントと方向づけのみ。本人が言語化する。

## 流れ

### Q1: 微分＝局所 / 積分＝大域 とは？
- ユーザーの回答：**「点と範囲の違い」**。微分は x=3 のような一点での性質、積分は x=0〜5 の範囲での性質。
- 評価：local/global の本質を一発で命中。
- ただし「一点での性質が**関数全体でどう変化していくか**」とも言った＝無意識に導関数の話を混ぜていた（後で回収する伏線に）。

### 微分が一点で測る「性質」とは？
- ユーザー：「関数の値が**どれくらい変化するか**」＝変化の勢い・変化率。正解。傾き＝変化の勢い。

### $f(x)=x^2$ の放物線を散歩する
- x=-2：下り / x=0：平ら（下り→上りの境目）/ x=3：上り、しかも x=-2 の下りより急。
- 全部正しい。左右対称でも中心から遠い x=3 の方が急、まで読めた。

### 傾きそのものが x で変わる → それは何？
- 「各 x に、その点の傾きを対応させる仕組み」を提示 → ユーザー：**「導関数」**。命中。

### 伏線回収
| ユーザーが Q1 で言ったこと | 名前 | 正体 |
|---|---|---|
| 一点での性質（変化の勢い） | 微分係数 $f'(a)$ | 一つの**数** |
| それが全体でどう変化するか | 導関数 $f'(x)$ | 一つの**関数** |
- ユーザーは教わる前に両方を同時にしゃべっていた＝直観が先回り。

### 直観と式の握手（締め）
- $f'(x)=2x$（結果だけ借用）に x=-2,0,3 を代入 → ユーザー：**-4, 0, 6**。
- 符号・大きさが散歩の「下り/平ら/急な上り」と完全一致。
  - -4＝下り、0＝平ら、6＝(4より大きく)より急な上り。
- 「$2x$ という関数に 3 を入れて 6 が出る」＝導関数に値を入れると微分係数が出る、も同時に体得。

## 効いたこと（方式の検証）
- AIは一度も答えを先出ししていない。結論は全部ユーザーが出した。
- 具体例（放物線の散歩）を当てた瞬間に抽象が軽くなった＝「大域→局所→例」の例。
- 言い間違い（混ざり）を否定せず伏線にして回収した。
- 辛さなし。テンポ・温度ともユーザー合意「ちょうどいい」。

## 次へのつなぎ
- ch02「微分係数と導関数」：今日散歩で体感した「微分係数＝数 / 導関数＝関数」を、
  $f'(a)=\lim_{h\to0}\frac{f(a+h)-f(a)}{h}$ の定義から積み上げる。授業の進度ともch02で一致。
